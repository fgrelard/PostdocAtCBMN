[[id:5693a566-f4ec-42c1-8932-bdcbd35d68e5][imzML]]

Specification for the file format.

The imaging binary data file (*.ibd) contains the mass spectral data of the MS imaging measurement.
The XML file (*.imzML) contains all relevant information about the MS imaging experiment (metadata + spectre position).
